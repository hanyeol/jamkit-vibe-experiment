# 100 Days - Jamkit 프로젝트

## TikTok Clone

Jamkit으로 만든 틱톡 스타일의 짧은 비디오 앱입니다.

### 주요 기능

1. **세로 스크롤 비디오 피드**
   - 위아래로 스와이프하여 다음/이전 영상 탐색
   - 자동 재생 및 반복
   - 풀스크린 비디오 플레이어

2. **소셜 인터랙션**
   - 좋아요 버튼 (하트 애니메이션)
   - 댓글 기능
   - 공유 기능
   - 팔로우/팔로잉

3. **사용자 프로필**
   - 프로필 사진 및 통계 (팔로워, 팔로잉, 좋아요)
   - 동영상 그리드 뷰
   - 좋아요한 영상 목록

4. **댓글 섹션**
   - 댓글 작성 및 좋아요
   - 답글 달기
   - 실시간 댓글 업데이트

### 파일 구조

```
100days/
├── catalog.yaml           # 메인 카탈로그
└── tiktok/
    ├── catalog.jmk        # 앱 진입점 및 데이터
    ├── video-feed.jmk     # 비디오 피드 UI
    ├── profile.jmk        # 사용자 프로필
    └── comments.jmk       # 댓글 섹션
```

### 필요한 아이콘 파일들

다음 아이콘들을 `tiktok/` 폴더에 추가하세요:
- `heart-icon.png` - 좋아요 버튼
- `comment-icon.png` - 댓글 버튼
- `share-icon.png` - 공유 버튼
- `music-icon.png` - 음악 아이콘
- `plus-icon.png` - 팔로우 버튼

### 실행 방법

1. Jamkit 앱을 설치합니다
2. 이 프로젝트 폴더를 Jamkit 앱에서 엽니다
3. "틱톡 클론" 앱을 선택하여 실행합니다

### 커스터마이징

- `catalog.jmk`의 `data` 섹션에서 비디오 데이터를 수정할 수 있습니다
- 각 `.jmk` 파일에서 스타일과 레이아웃을 변경할 수 있습니다
- 색상은 틱톡의 브랜드 컬러 `#FE2C55`를 사용합니다
