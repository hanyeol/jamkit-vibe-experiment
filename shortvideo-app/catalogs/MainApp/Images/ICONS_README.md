# Required Icons for ShortVideo App

이 앱에 필요한 아이콘 이미지들입니다. 각 아이콘은 @m (medium), @u (ultra), @x (extra) 해상도로 제공되어야 합니다.

## Action Buttons

### 1. Heart Icon (좋아요)
- `icon_heart.png` - 빈 하트 아이콘 (좋아요 안한 상태)
- `icon_heart_filled.png` - 채워진 하트 아이콘 (좋아요 한 상태)
- 색상: 흰색 또는 투명 배경에 흰색
- 크기: 50x50 기준

### 2. Comment Icon (댓글)
- `icon_comment.png` - 말풍선 아이콘
- 색상: 흰색
- 크기: 50x50 기준

### 3. Share Icon (공유)
- `icon_share.png` - 공유 화살표 아이콘
- 색상: 흰색
- 크기: 50x50 기준

### 4. Record Icon (녹화/업로드)
- `icon_record.png` - 플러스(+) 또는 카메라 아이콘
- 색상: 흰색
- 크기: 60x60 기준

## Bottom Navigation Icons

### 1. Home Icon
- `icon_home.png` - 홈 아이콘
- `icon_home_active.png` - 활성화된 홈 아이콘
- 크기: 30x30 기준

### 2. Discover Icon
- `icon_discover.png` - 검색/발견 아이콘
- `icon_discover_active.png` - 활성화된 검색 아이콘
- 크기: 30x30 기준

### 3. Create Icon
- `icon_create.png` - 생성 아이콘 (중앙의 큰 버튼)
- 크기: 40x40 기준

### 4. Inbox Icon
- `icon_inbox.png` - 메시지 아이콘
- `icon_inbox_active.png` - 활성화된 메시지 아이콘
- 크기: 30x30 기준

### 5. Profile Icon
- `icon_profile.png` - 프로필 아이콘
- `icon_profile_active.png` - 활성화된 프로필 아이콘
- 크기: 30x30 기준

## 임시 해결책

아이콘이 없는 경우, 텍스트 라벨로 대체하여 앱을 테스트할 수 있습니다.
실제 배포 시에는 디자이너가 제작한 아이콘을 사용하세요.
